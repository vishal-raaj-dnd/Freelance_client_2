## About

Dawoodiya Masjid & Madrasah is an institution deeply rooted in classical Islamic scholarship, offering a comprehensive Dars-e-Nizami curriculum alongside dedicated Hifz & Tajweed tracks. This repository hosts the official website built with React, Vite, and TailwindCSS.

## Tech Stack

- **React 19** — UI framework
- **Vite** — Build tooling
- **TailwindCSS v4** — Styling
- **Motion (Framer Motion)** — Animations
- **React Router v7** — Client-side routing

## Run Locally

**Prerequisites:** Node.js (v18+)

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000)

## Deployment

This project is deployed on **Vercel**. Push to the `main` branch to trigger auto-deployment.
